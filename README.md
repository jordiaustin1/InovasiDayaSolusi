# InovasiDayaSolusi
Jawaban untuk Test Internship

# How to Use
1. Import database:
   mysql -u root viewdata_db < viewdata.sql

2. Jalankan API:
   dotnet restore
   dotnet run --project ViewDataApi.csproj

3. Akses endpoint:
   http://localhost:5050/api/viewdata

4. Dokumentasi Swagger:
   http://localhost:5050/swagger
